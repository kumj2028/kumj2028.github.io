Classical Cipher Tool

Substitution Cipher

A monoalphabetic cipher with 26! possible keys – but still breakable.

Use this page to experiment with a general substitution cipher: generate a random key, encrypt your own messages, study letter frequencies, and iteratively refine a key guess to recover the plaintext.

Encrypt

Apply a monoalphabetic substitution to your plaintext.

The key defines where each letter A–Z goes. For example, if the first letter is Q, then A → Q; if the second is W, then B → W; and so on. Letters must not repeat.

Decrypt (Frequency Analysis)

Analyze letter frequencies, propose a key, and iteratively improve it.

This key uses the same format as above (26 letters, A–Z mapped in order). Adjust it as you recognize common patterns (E, THE, AND, etc.).
